# Hangman Game 🎮

Console-based Hangman game built in Python as part of the CodeAlpha Python Programming Internship (Task 1).

## About
This is a simple command-line implementation of the classic Hangman game where the player has to guess a hidden word letter by letter within a limited number of attempts.

## Features
- Random word selection
- Limited number of guesses
- Input validation for repeated/invalid guesses
- Win/lose message display

## Technologies Used
- Python 3
