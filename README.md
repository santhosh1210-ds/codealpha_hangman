# 🎮 Hangman Game

## 📌 Project Overview
Console-based Hangman game built in Python as part of the CodeAlpha Python Programming Internship (Task 1). The player has to guess a hidden word letter by letter within a limited number of attempts.

## 🛠️ Tools & Libraries
- Python 3
- `random` module

## 📂 Project Structure
- Word list for random selection
- Attempt/lives counter
- Input validation logic

## 🔍 How It Works
1. A random word is selected from the word list
2. Player guesses one letter at a time
3. Correct guesses reveal letter positions
4. Wrong guesses reduce remaining attempts
5. Game ends on win (word guessed) or loss (attempts exhausted)
